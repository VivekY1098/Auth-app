# Contributing to Ollion Projects

Thank you for contributing to Ollion's projects!

## Reporting Bugs
- Open an issue with a clear description and steps to reproduce.

## Suggesting Features
- Discuss in an issue before submitting code.

## Submitting Pull Requests
1. Fork the repository.
2. Clone your fork: `git clone <url>`.
3. Create a branch: `git checkout -b feature-name`.
4. Make changes and commit: `git commit -m "Add feature"`.
5. Push to your fork: `git push origin feature-name`.
6. Open a pull request with a detailed description.

## Development Setup
- Follow project-specific instructions in README.md
